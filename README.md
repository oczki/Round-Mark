# Round-Mark
Userscript to place tiny floating bookmarks on any website.

![image](https://user-images.githubusercontent.com/2924029/99459060-c7f63280-292d-11eb-92d1-e0629a45c1d4.png)

Installation
---

Get a browser extension that lets you run userscripts. Depending on your browser it might be called Violentmonkey, Greasemonkey, or some other animal.

Then, open the `RoundMark.user.js` file in this repository and click the `Raw` button on GitHub. The extension should ask you whether you want to install the userscript.

Alternatively, copy the contents of the `RoundMark.user.js` file and paste it in your extension's editor.

Usage
---

| Hotkey                                             | Action                                     |
|----------------------------------------------------|--------------------------------------------|
| <kbd>Alt</kbd> + <kbd>double click</kbd> on a page | Place a mark                               |
| <kbd>Alt</kbd> + <kbd>scroll up</kbd>              | Scroll to a bookmark above the current one |
| <kbd>Alt</kbd> + <kbd>scroll down</kbd>            | Scroll to a bookmark below the current one |
| <kbd>Alt</kbd> + <kbd>double click</kbd> on a mark | Delete that mark                           |
| <kbd>Alt</kbd> + <kbd>middle click</kbd>           | Delete all marks                           |
